# homebrew-tap

Homebrew tap for [toaweme](https://github.com/toaweme) tools.

## Install

### Care
```sh
brew install toaweme/tap/care
```

### Blink
```sh
brew install toaweme/tap/blink
```

Formulas in `Formula/` are generated and pushed automatically by each project's release pipeline ([GoReleaser](https://goreleaser.com)). Do not edit them by hand.
